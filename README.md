# Kyouka

注入到网页中的Web小工具合集 不乏利用各种冷门API

### 安装

在Release中下载扩展程序的crx文件 打开Chrome扩展页面 将文件拖动到页面中即可安装

### 功能列表
- 自由编辑页面
- 启用所有元素
- 移除输入限制
- 显示隐藏元素 
- 强制翻译元素内容
- 鼠标中键移除元素
- 强制在新标签页打开超链接
- 强制RTL(右到左)布局
- 关闭拼写检查
- 注入JS
- 注入CSS
- 屏蔽eval函数
- 屏蔽open函数
- 屏蔽sendBeacon函数
- 屏蔽日志输出
- 持久化Cookie
- 强制属性可配置
- 打印页面JSON操作(stringify parse)
- 屏幕像素取色
- 将页面转为文档画中画
- 录制Canvas
- 录屏
- 播放音频
- 修改网页背景
- 修改网页图标
- 修改网页标题
